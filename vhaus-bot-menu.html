What the bot does
📷
Submit a Sales Order
Send photo → AI reads → confirm → saved
1
Take a clear photo of the handwritten sales order and send it to the bot
2
Bot extracts all fields with AI — SO number, customer, items, delivery date, balance
3
Review the preview — check all details are correct
4
Reply to confirm or make corrections
YES — save order
CANCEL — discard
or type a correction
Correction examples
"delivery date is October"
"balance is 3840"
"item 2 is king size"
"remove item 3"
"add sofa 3 seater"
📅
Update Delivery Date
Text the bot to reschedule an existing order
1
Type a message in this format:
SO: 31074
DELIVERY DATE: 10/6
optional note on next line

Check Delivery Schedule
See all orders grouped by area for a date
1
Type the /schedule command followed by a date
2
Bot replies with all orders for that day, grouped by area, with balances highlighted
/schedule 15/7
/schedule 2026-07-15
Tips for best results
💡
Good photo = fast extract
Lay the order flat, good lighting, no shadows covering the SO number or delivery date
⚠️
One draft at a time
Confirm or cancel current order before sending another photo — otherwise replies go to the active draft
🔴
Balance in red
If balance shows red in the delivery sheet, it means payment is outstanding — double check before dispatch
🔁
Duplicate SO blocked
Bot won't allow the same SO number to be saved twice — draft is auto-discarded if SO already exists
